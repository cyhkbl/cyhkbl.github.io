学生薅羊毛指南

Github Student Benefits

学信网下载学籍验证报告并翻译

进入https://my.chsi.com.cn/archive/bab/index.action，下载学籍验证报告。

使用学信网档案英文版转化器进行翻译，生成英文版学籍报告。

在 General full-time 后加上 remote study 解决定位问题。

统一个人信息

将Payment Information、Public Profile中的姓名与学籍报告中的信息保持一致。注意需要全英文。

Bio可以改为student of xxx university @ xxx university。

Company可以改为xxx university。

申请

使用手机，关闭VPN和代理，使用学校邮箱登录Github Student Benefits页面，点击”Get your pack”按钮，进入申请页面。

使用手机拍摄电脑上的英文学籍报告。在拍照时，如果无法切换后置摄像头，可以先锁屏，再开启手机。

如果无法通过，可能需要打开Github账户双因素认证。

Azure for Students

进入https://azure.microsoft.com/en-us/free/students，点击”Activate now”按钮，使用学校邮箱注册或用Github Student Benefits认证，即可获得$100的Azure信用额度和部分免费服务。

如果Azure没有通过，可以在审核页面选择“与Microsoft交谈”，创建一个支持工单，标题写“无法获得学生权益”。大约两三天之后会有工作人员给你发邮件，给你一个验证代码。

Google AI Pro

首先进入这个https://support.google.com/googleone/answer/16417758?hl=zh-Hant&visit_id=639005046644967440-2850062761&p=aip_student_trial&rd=1页面，点击“去Google One验证”进入到验证页面。

然后点击“验证学生身份”，将SheerID网址复制到https://batch.1key.me/过验证，打开验证后网址，绑定银行卡即可。

理论上，在已经认证完了就差支付的阶段，先不点，去手机上挂新加坡节点绑上银联卡，绑完之后再通过电脑刚刚打开的页面支付，这时候直接可以用绑的银联卡付款。

AWS Educate

  1. 打开 AWS Educate 页面。
  2. 点击“Join AWS Educate”按钮并填写申请表格。
  3. 使用 edu 邮箱完成验证。

Autodesk 教育版软件

  1. 访问 Autodesk Education Community 页面。
  2. 创建 Autodesk 账户并选择“教育版”选项。
  3. 上传证明文件以确认学生身份。

JetBrains 学生包

  1. 前往 JetBrains Student License 页面。
  2. 点击“Apply Now”，登录或创建 JetBrains 账户。
  3. 使用 edu 邮箱验证身份，或者上传在读证明材料。

Notion 学生计划

  1. 前往 Notion for Students and Educators 页面。
  2. 注册 Notion 帐号并输入您的 edu 邮箱地址完成验证。

Canva Pro 教育版

  1. 前往 Canva 教育计划 页面。
  2. 点击“注册”或“申请教育版”按钮。
  3. 使用您的 edu 邮箱地址创建账户，或者登录现有账户。
  4. 上传学生证或学校开具的在读证明以完成身份验证。

Coursera 和 Udemy 学习平台

  1. 打开 Coursera for Students 页面。
  2. 创建账户并填写个人信息，使用 edu 邮箱完成注册。
  3. 浏览课程目录并选择符合条件的免费或优惠课程进行学习。

Amazon Prime 学生优惠

  1. 进入 Amazon Prime 学生页面。
  2. 使用 .edu 邮箱注册。
  3. 提供学生身份信息进行验证。
  4. 完成注册后可享受学生优惠。

阿里云学生计划

  1. 访问 阿里云高校计划 页面。
  2. 认证方式为使用支付宝学信网认证。
  3. 认证后一年300元的直抵用优惠券
  4. 审核通过后即可享受优惠。

Sketch

  1. 前往 Sketch Education Store 页面。
  2. 创建或登录 Sketch 帐户。
  3. 填写申请表格并提供学生身份验证信息（如 edu 邮箱或其他证明材料）。
  4. 验证通过后即可获得一年的免费许可证。

Ansys Academic Program (Education Discount)

  1. 访问 Ansys for Students 页面。
  2. 点击“Free Student Software Downloads”按钮，进入学生版软件下载页面。
  3. 创建或登录 Ansys 帐户。
  4. 填写表格并上传相关证明文件（如 edu 邮箱或学生证）。
  5. 下载并安装软件后，即可激活教育许可。

CST Studio Suite (Education Discount)

  1. 访问 3DEXPERIENCE Edu 页面。
  2. 免费下载
  3. 使用Edu邮箱注册账户验证后进入software3ds。
  4. 选择免费下载即可。
  5. 免费的有CATIA ICEM Surf ，CATIA CATIA Magic/No Magic，SIMULIA SIMULIA Abaqus，SIMULIA SIMULIA CST。
