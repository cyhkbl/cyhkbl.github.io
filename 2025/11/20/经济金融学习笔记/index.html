经济金融学习笔记

股票市场

常见概念

一级市场： 创造和首次发行商品的市场。公司是卖家，资金直接从投资者流向公司。如IPO（首次公开发行）、发、私募。

二级市场： 买卖这些已经发行了的商品的市场。投资者之间相互交易，资金在投资者之间流动。

开盘价： 是集合竞价的结果，由开盘前一段时间内所有人的买卖委托共同决定。在A股市场，是在上午9:15 - 9:25*的时段。所有高于这个价格的买入委托和所有低于这个价格的卖出委托**，都会以这个开盘价成交。等于这个格的委托，则按“时间优先”或“数量优先”的原则部分或全部成交。

收盘价： 是连续竞价最后一刻的价格，或者是最后一段时间的加权平均价。目前A股市场的收盘价在下午14:57 -15:00采用 “集合竞价” 方式产生。在连续竞价结束（14:57）后，系统进入3分钟的收盘集合竞价时段。系统会收集在这分钟内所有的买卖委托，并按照 “最大成交量” 原则计算出收盘价，并进行收盘集合竞价的撮合成交。

例如：

假设某只股票在集合竞价时，有以下几个出价：

委托方向 委托价格（元） 委托数量（股）
买入 10.20 1000
买入 10.10 2000
买入 10.00 3000
卖出 10.00 2000
卖出 10.10 2000
卖出 10.20 1000

系统会尝试不同价格下的可成交量：

在 10.00元：买入委托（≥10.00）有 1000+2000+3000 = 6000股；卖出委托（≤10.00）有2000股。可成交量为 2000股。

在 10.10元：买入委托（≥10.10）有 1000+2000 = 3000股；卖出委托（≤10.10）有2000+2000 = 4000股。可成交量为 3000股。

在 10.20元：买入委托（≥10.20）有1000股；卖出委托（≤10.20）有2000+2000+1000 = 5000股。可成交量为 1000股。

比较发现，在 10.10元 时，可成交量最大（3000股）。因此，10.10元 就会被确定为当天的开盘价。

除息(XD)： 公司向股东派发现金红利。

举例：股价10元，每股派现1元。除息日开盘参考价变为9元。你的总资产不变：股票市值9元 + 现金1元 = 10元。

除权(XR)： 公司向股东送红股或转增股本。除权价 = 股权登记日收盘价 / (1 + 每股送转股比例)

举例：股价20元，实行“10送10”（每10股送10股）。除权日开盘参考价变为 20 / (1 + 1) = 10元。你持有的股票数量翻倍，但每股价格减半，总市值不变。

DR： 同时除权除息。

送股： 公司将利润转化为股本，按比例免费送给股东（如10送3）。资金来源：净利润。

转股： 公司将资本公积金转化为股本，按比例免费转给股东（如10转5）。资金来源：资本公积（资本（股本）溢价）。

总股本： 发行的所有股票的总数量。公司的总市值 = 股价 × 总股本。

保证金： 借钱买股票（融资）或借股票来卖出（融券）时，需要向券商缴纳保证金。在期货/期权交易开仓时，交易所和期货公司会要求您存入一定比例的资金作为履约保证。

流通股： 在证券交易所市场上可供投资者公开交易的股票数量。流通股多，则股价相对稳定。通常认为流通市值500亿人民币以上或流通股数量在几十亿股甚至上百亿股的股票为大盘股/蓝筹股。

市盈率： 股价 / 每股收益。PE为10，意味着你投资这家公司需要10年才能回本。

市净率： 股价 / 每股净资产。净资产 = 总资产 - 总负债。PB为1，意味着你正好以公司的净资产价值买入。PB越高，说市场给予其净资产的价值溢价越高。

每股收益： 公司税后利润 / 总股本。EPS直接反映了公司的盈利水平。持续增长的EPS通常是公司成长性的良好体现。

净资产收益率： 税后利润 / 净资产。衡量的是公司利用股东投入的资本（净资产）来创造利润的能力。

股息率： 每股股息 / 股价。

财报（年报、季报）: 包括资产负债表、利润表、现金流量表。获取渠道：巨潮资讯网等。对财报的分析可以借助AI工具进行。

基本面分析

股票的价值由其背后公司的内在价值决定。通过分析公司及其所处环境来判断其“真实价值”并在市场价格低于价值时买入，高于价值时卖出。

分析角度

  1. 宏观经济：

  • 经济增长（GDP）： 强劲的GDP增长通常利好股市（企业盈利预期好），但若增长过热引发通胀和紧缩担忧，则可能变为利空。

    • 看增速： 看同比增速、环比增速、增速的加快/放缓、增速与市场预期的对比。

    • 看结构： 分析增长由什么驱动。

      消费反映了内需的健康程度。

      投资可以重点关注固定资产投资（房地产、基建、制造业），与固定资产投资直接相关的板块一定会受益。市场内生动力造成的制造业投资和民间投资通常利好股市，而政府逆周期调节（包括基础设施投资和房地产投资）对股市影响偏空。

      净出口可以关注对外贸易是顺差还是逆差，对经济增长是正贡献还是负贡献？

    • 看位置： 处于经济周期的阶段（复苏、扩张、过热、衰退）。

  • 资金成本（利率）： 利率低，资金便宜，有利于股市和债市；利率高，资金昂贵，有利于现金类资产。

    • 看央行的政策利率：这是市场的“锚”。
    • 看利率走廊：分析短期利率（如SHIBOR、回购利率）在利率走廊中的位置。
    • 看收益率曲线：短期和长期国债收益率之间的利差。一条陡峭的曲线（长端利率远高于短端）通常预示经济前景乐观；一条平坦甚至倒挂的曲线则可能预示经济衰退。

市场影响：

  • 利率是资产定价的基石。所有资产的现值都是其未来现金流的贴现，贴现率就与利率密切相关。

  • 股市：利率下行，估值提升，利好成长股和高估值板块；利率上行，估值承压，利好银行、保险等板块。

  • 债市：利率与债券价格呈负相关。加息周期，债市走熊；降息周期，债市走牛。

  • 汇率：一国利率上升，会吸引国际套利资金流入，推动本币升值。

  • 通货膨胀 - CPI & PPI： 温和通胀利好股市，恶性通胀或通缩则利空股市。

    • 消费者物价指数（CPI）：衡量居民购买的一篮子商品和服务的价格变化，反映下游消费端的通胀压力。

    • 工业生产者出厂价格指数（PPI）：衡量工业企业产品出厂价格变化，反映中上游生产端的通胀压力。

    • 看趋势和水平： CPI和PPI是持续上行还是下行？是否处于央行设定的目标区间之内或之上？

    • **看传导：**PPI向CPI的传导是否顺畅？如果PPI高而CPI低，说明终端需求不足，企业利润被挤压。

    • 核心CPI： 剔除食品和能源价格后的CPI。

4. 货币政策

是什么：央行通过控制货币供应量和利率来调节经济的工具。

如何分析：

  • 政策取向：是“宽松”、“稳健”、“紧缩”还是“中性”？
  • 工具使用：
    • 价格型工具：调整利率（加息/降息）。
    • 数量型工具：调整存款准备金率（降准/升准）、公开市场操作（OMO、MLF、SLF等）。
  • 看央行沟通：关注央行会议纪要、声明和官员讲话，特别是对通胀和经济增长的前景判断，这暗示了未来的政策方向。

市场影响：

  • 货币政策是股市和债市最直接的驱动力之一。宽松的货币政策通常催生牛市，紧缩的货币政策则往往导致市场下跌。

5. 财政政策

是什么：政府通过税收和公共支出调节经济的工具。

如何分析：

  • 看财政赤字率：赤字率扩大通常意味着政府正在实施扩张性财政政策（减税或增支），以刺激经济。
  • 看专项债发行：特别是用于基建的专项债，是观察政府投资力度的重要窗口。
  • 看政策方向：是“积极”（扩张）还是“稳健”（中性或略微收缩）？

市场影响：

  • 股市：扩张性财政政策直接利好基建、建材、工程机械等相关行业，并通过乘数效应刺激整体经济，间接利好股市。
  • 债市：扩张性财政政策意味着政府需要发行更多国债，增加债券供给，可能对债市形成压力（“挤出效应”）。同时，刺激经济可能推高利率，也利空债市。

6. 汇率

是什么：一国货币与另一国货币的比率。

如何分析：

  • 看趋势：本币是处于升值通道还是贬值通道？
  • 分析原因：是源于利率差异（套息交易）、贸易顺逆差、还是资本流动？
  • 与货币政策的关系：央行货币政策常常需要在“保汇率”和“稳增长”之间权衡。例如，在资本外流、本币贬值压力大时，央行降息的空间会受到制约。

市场影响：

  • 股市：
    • 本币升值：利好拥有大量外债的行业（如航空），利好以本币计价的资产（吸引外资流入）。但利空出口型企业（产品变贵，竞争力下降）。
    • 本币贬值：利好出口型企业，但可能引发资本外流，对股市整体形成压力。
  • 债市：本币贬值可能引发资本外流，导致国内流动性收紧，利空债市。

7. 国际形势

是什么：全球地缘政治、经济、贸易环境。

如何分析：

  • 全球主要经济体状况：美国、欧洲的经济周期和货币政策。尤其是美联储的政策，对全球资本流动有巨大影响。
  • 全球贸易关系：是否存在贸易摩擦、关税壁垒？全球供应链是否顺畅？
  • 地缘政治风险：如战争、冲突、制裁等，会影响全球能源和粮食价格，进而影响全球通胀。
  • 大宗商品价格：原油、铜等关键商品的价格变化，是全球需求的晴雨表。

市场影响：

  • 风险偏好：国际形势稳定向好，提升全球投资者的风险偏好，资金流向风险资产（股市）；形势紧张则提升避险偏好，资金流向黄金、美元、美债等避险资产。
  • 输入性通胀/通缩：通过影响大宗商品价格，传导至国内。

综合联动分析：一个典型的场景

假设当前情况：GDP增速放缓，但CPI通胀持续超预期。

  1. 核心矛盾：出现了“类滞胀”风险（经济增长停滞 + 通货膨胀）。
  2. 央行困境：央行面临两难。
    • 如果要保增长，应该降息降准（宽松）。
    • 但要控通胀，应该加息（紧缩）。
  3. 政策预测：此时央行通常会更加侧重于控通胀，因为通胀失控的长期危害更大。因此，货币政策可能会保持“紧”或“中性”，不会轻易放松。
  4. 市场影响：
    • 股市：承压。因为“经济差”损害企业盈利，“货币紧”压制估值。市场表现可能较差。
    • 债市：同样承压。虽然经济差利好债市，但高通胀和紧缩的货币政策是债市更强大的敌人。
    • 结论：这是一个对股债都不太友好的宏观环境。

GDP、利率、通胀、货币政策、财政策、汇率、国际形势等。

行业分析： 行业生命周期（初创、成长、成熟、退）、市场规模、竞争格局（波特五力模型）、行业策、技术发展趋势等。寻找有前景的“赛道”。
公司分析（重中之重）：
业务与商业模式： 公司做什么？怎么赚钱？核心竞争力（护城河）是什么？（品牌、技术、成本、规模、网络效应、特许经营权等）
管理层： 管理团队的能力、诚信、过往业绩、战略眼光。
财务状况（财报解读）：
资产负债表： 看家底（资产、负债、所有者权益），关注偿债能力（流动比率、速动比率、资产负债率）、资产质量。
利润表： 看盈利能力（营业收入、毛利率、净利率、营业利润率、净利润、每股收益），关注盈利的持续性和增长性。
现金流量表： 看“真金白银”（经营、投资、筹资活动现金流），关注经营现金流是否健康，公司造血能力如何。
估值分析： 判断当前股价相对于公司内在价值是贵还是便宜？常用指标：
市盈率： 股价/每股收益。反映市场对公司未来盈利的预期。注意区分静态PE、动态PE（预测）、滚动PE（TTM）。不同行业、不同成长阶段的公司PE差异很大，要横向（同行业）和纵向（自身历史）比较。
市净率： 股价/每股净资产。常用于评估重资产或周期性公司，银行股常用。同样需要比较。
市盈率相对盈利增长比率： PEG = PE / (预期盈利增长率*100)。一般认为PEG<1表示估值相对成长性可能偏低。
股息率： 每股股息/股价。对看重现金分红的投资者重要。
自由现金流折现模型： 更复杂的绝对估值方法，预测公司未来产生的自由现金流，并折现回当前价值。需要较多假设。
优点： 逻辑清晰，有助于理解公司本质，适合中长投资决策。
缺点： 信息获取和分析门槛较高，短期市场波动可能偏离基本面。

债券市场

当然可以，这是一个非常好的问题。下面我将为您详细解释债券市场，以及个人如何参与投资。

第一部分：什么是债券市场？

简单来说，债券市场是发行和交易债券的场所。

您可以把它想象成一个巨大的“借贷市场”：

  • 借款人（发行方）：需要资金，例如政府、地方政府、企业、金融机构等。他们通过发行债券来向大众借钱。
  • 贷款人（投资者）：有闲置资金，希望通过出借资金来获得稳定收益。他们购买债券，成为债权人。

债券的核心要素：

  1. 面值：债券的票面价格，通常是到期后发行人偿还给你的本金。
  2. 票面利率：发行人承诺每年支付给你的利息比率。例如，一张面值1000元、利率5%的债券，每年会付给你50元利息。
  3. 到期日：债券的最终偿还日期，到了这一天，发行方会将本金（面值）还给你。

债券市场根据交易阶段分为：

  • 一级市场：发行新债券的市场，投资者在这里直接认购新发行的债券。
  • 二级市场：已发行债券的交易市场，投资者可以像买卖股票一样，相互买卖已经上市的债券。

第二部分：个人可以到债券市场投资吗？

完全可以！

个人投资者不仅可以，而且债券是构建多元化投资组合、降低整体风险的重要工具。与股票市场的高波动性相比，债券通常能提供更稳定的收入流，并起到“资金避风港”的作用。

第三部分：个人通过什么方式投资债券？

个人投资者有多种渠道可以进入债券市场，主要分为直接投资和间接投资两大类。

方式一：直接投资

这是指个人直接购买债券，成为债券的持有人。

  1. 证券交易所（场内市场）

    • 如何操作：通过你的股票账户，像买卖股票一样，在交易软件中输入债券代码进行买卖。
    • 可投资品种：
      • 国债：由国家发行，信用等级最高，几乎无违约风险。在交易所交易的国债被称为“记账式国债”。
      • 地方政府债：由地方政府发行，信用等级也很高。
      • 金融债：由银行或金融机构发行。
      • 公司债/企业债：由上市公司或大型企业发行，收益率通常高于国债，但风险也相应更高。
    • 优点：流动性好，可以随时买卖；门槛较低，通常以“手”（10张）为单位交易。
    • 缺点：需要自己研究发行主体的信用状况；价格会随市场利率波动，可能有价差亏损的风险。

  2. 银行柜台（场外市场）

    • 如何操作：携带身份证去银行网点，或通过手机银行、网上银行购买。
    • 可投资品种：
      • 储蓄国债：这是专门为个人投资者设计的国债，不可上市交易，但通常可以提前兑取。分为凭证式（纸质凭证）和电子式两种。这是最受普通投资者欢迎的方式之一，因其安全、收益稳定、购买方便。
    • 优点：非常安全，操作简单直观，收益稳定。
    • 缺点：流动性相对较差，提前兑取可能会有利息损失；发行次数有限，需要抢购。

方式二：间接投资

这是指个人通过购买投资于债券的金融产品来参与市场，将专业的事情交给专业机构。

  1. 债券型基金

    • 如何操作：通过银行、证券公司、支付宝、天天基金等第三方平台购买。
    • 原理：基金公司把众多投资者的钱聚集起来，由专业的基金经理去投资一个包含多种债券的组合。
    • 优点：
      • 专业管理：无需自己挑选债券。
      • 分散风险：投资一篮子债券，避免“把鸡蛋放在一个篮子里”。
      • 门槛极低：几十元或几百元即可起投。
      • 流动性好：开放式基金可以随时申购赎回。
    • 类型：
      • 纯债基金：只投资债券，风险相对较低。
      • 一级债基/二级债基：可以少量参与新股申购或投资股票，风险和潜在收益都更高。

  2. 银行理财产品

    • 很多银行理财产品（尤其是中低风险等级的产品）其主要投资方向就是债券市场。你可以通过银行App或网点购买这些产品。

总结与建议

投资方式 渠道 优点 缺点 适合人群
储蓄国债 银行柜台/网银 最安全、收益稳定、门槛低 流动性较差、需要抢购 风险厌恶型，追求绝对安全的投资者
交易所债券 证券公司/股票账户 流动性好、品种丰富 需要专业知识、价格有波动 有一定投资经验，愿意花时间研究的投资者
债券基金 基金公司/第三方平台 门槛低、专业管理、分散风险 有基金管理费、净值有波动 最适合绝大多数个人投资者的方式，省心省力
银行理财 银行 操作方便、风险相对可控 收益不透明、可能有手续费 习惯在银行进行资产配置的投资者

给新手的建议：

  1. 从最安全的开始：如果想直接投资，可以从储蓄国债开始尝试，感受一下债券投资的流程。
  2. 首选债券基金：对于绝大多数没有时间和专业知识去研究单个债券的投资者来说，债券型基金（尤其是纯债基金） 是最佳选择。它完美地平衡了风险、收益和便利性。
  3. 注意风险：债券并非完全没有风险。主要风险包括：
    • 利率风险：市场利率上升，已发行债券的价格会下跌。
    • 信用风险：发行人可能还不上钱（违约）。
    • 流动性风险：某些债券可能不容易快速卖出。

希望这个详细的解释能帮助您全面了解债券市场和个人投资方式！

强劲的GDP增长通常利空债市（经济好，通胀和利率上行压力大，债券价格下跌）。
