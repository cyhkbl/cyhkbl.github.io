scoop-windows的应用商店

虽然Windows最近也推出了自己的软件商店——Microsoft Store，我却本着不把文件放C盘、支持开源等多个理由，在windows上使用最为著名的scoop来对自己的软件进行管理。

首先，需要设置PowerShell执行策略，通过输入以下命令Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser即可。

Invoke-Expression (New-Object System.Net.WebClient).DownloadString('https://get.scoop.sh')

这样，scoop就安装完成了。

scoop配置

有时，我们需要通过代理下载一些软件，输入以下命令改变下载时使用的代理（端口号与自己使用的一致）：

scoop config proxy 127.0.0.1:7890

或找到Scoop配置文件，路径C:\Users\username\.config\scoop\config.json，然后直接修改里面的配置。

scoop命令

Scoop的操作命令基本结构是scoop + 动词 + 对象，比如安装typora，通过输入scoop install typora即可自动完成软件的官网进入+下载+安装等操作。

以下是一些常用的命令说明：

cleanup *——删除全部旧版本应用。

scoop cache rm *——删除全部缓存。

search——搜索仓库中是否有相应软件。

install——安装软件。

uninstall——卸载软件。

update——更新软件。可通过scoop update *更新所有已安装软件，或通过scoop update更新所有软件仓库资料及Scoop自身而不更新软件。

hold——锁定软件阻止其更新。

info——查询软件简要信息。

home——打开浏览器进入软件官网。

如果忘记了，可通过输入scoop help来查询语法，以及更多不怎么常用的操作指导。

推荐软件仓库

Scoop添加软件仓库的命令是scoop bucket add bucketname(+ url可选)。如添加extras的命令是scoop bucket add extras，执行此命令后会在scoop文件夹中的buckets子文件夹中添加extras文件夹。

Scoop官方有一些仓库可供使用：

main - Default bucket for the most common (mostly CLI) apps
extras - Apps that don’t fit the main bucket’s criteria
games - Open source/freeware games and game-related tools
nerd-fonts - Nerd Fonts
nirsoft - Almost all of the 250+ apps from Nirsoft
java - A collection of Java development kits (JDKs), Java runtime engines (JREs), Java’s virtual machine debugging tools and Java based runtime engines.
nonportable - Non-portable apps (may require UAC)
php - Installers for most versions of PHP
versions - Alternative versions of apps found in other buckets

除了官方的软件仓库，Scoop也支持用户自建仓库并共享，其中dorado仓库里面有许多适合中国用户的软件。添加dorado仓库的命令如下：scoop bucket add dorado https://github.com/chawyehsu/dorado。

此外，若多个仓库间的软件名称冲突，可以通过在软件名前添加仓库名的方式避免冲突，如scoop install dorado/appname。

化竞。

这篇文章，我在初赛前就准备写了。

在开头，我应该说我不是一个“正牌”竞赛生。目前竞赛的趋势是低龄化，我们年级正常搞化竞的都是在高一前的暑假加入团队的，而低一级的还有更早的，也就是通过“首创班”这个半违规渠道，提前一年开始学习各种竞赛。而我在高一上的寒假，才开始踏入化竞的大门。这个时候有很多人已经基本结束了初赛的第一轮学习了（基本是我去考初赛时的水平）。

我当时想去搞竞赛，最初目的也很简单：想补一补课内的化学。看到我们班的各科竞赛生在这门课都不同程度的发生“偏科”（褒义），我心里想的是学什么补什么。当然第一学期我的化学确实也是一团糟，我后来想想深层原因还真不是我的问题。现在的化学教育明显是有缺陷的，基本的化学从初二才开始，位居主要学科倒数第一。要知道，物理和生物都是从小学三年级就开始的。还有一个原因，是我不太去外面补习班上课，估计寒假非常空虚，疫情又迟迟不结束，整个人处于一种浑浑噩噩的状态。不管是什么原因，我厚着脸皮去找了球哥，就这样开始了我短暂的化竞生涯。

当然，我总是要“欲扬先抑”一下。说实话，竞赛对学生，是另一种消耗。竞赛的保送、强基A+固然诱人，但是其代价也是巨大的。我亲眼见证我们学校生物竞赛的同学接近2个月没有回班上课、计算机竞赛“永久性”停课对学生造成的伤害有多大，更甚者必须留级重读。化竞几乎不停课，但偶尔几次停课我也必须溜回教室，不可谓不惊险。而且往往越能出成绩的教练，停课越多。其次，竞赛确实是极度压榨时间的。参加后我才真正懂得了竞赛是学有余力同学的游戏，而不是普通人改变命运的方式。说实话，高考乃至中考其实已经到了要拼智商的地步了，竞赛亦然。竞赛就是一个年级七八十个学生，给那三四个未来可能可以进省队进国集的同学垫垫底而已，好让他们感受到虐菜的乐趣。还有一点，时间和课内成绩，只能选择一个。这一点在我这个搞竞赛的同时奉行“周末享乐主义”的人眼中是至理至真的言论，因为我自己就遭到了课内成绩带给我的暴击。

寒假里，让我大为惊讶的是，我们并没有像之前的学生一样花一段时间去学高中内容，而是用3天左右学完了选修二、三的主干知识。这种知识密度和知识量是我从未感受过的。之后我们直接“跟”上了早一批的同学，开始学习大学化学的内容。从现在来看，这明显是囫囵吞枣了。我还清楚地记得，在我在判断醛、酮、醚等都要想半天的时候，我就在喧闹的教室里，看着球哥画着机理，听着同学讲着LUMO、HOMO，这给我小小的心灵留下了大大的震撼。

开学后，直接开始的是预赛的复习。当一叠叠的预赛真题和模拟题摆在我眼前时，我对高考化学的羸弱暴露无遗。预赛大部分都是高考内容，我当时却不知道吃错了什么药，一直在看Clayden（当然对于之后还是有帮助的）；结果在预赛前成绩一直没有提升，对高中化学也没有一个系统的认识。在考前每周三晚自修球哥还会给我们几个“弱者”开小灶，我很感谢球哥。

预赛在我的脑海里并没有很长时间，且大部分时间都在刷题。由于学校有大量的保送名额，很多人也没当回事。对于初赛，我们学校似乎更加注重有机。其实在竞赛低龄化的前提下，高二开始搞有机是正常现象。但是对于高中开始搞竞赛甚至像我这种才搞了两三个月的人来说，我很后悔没有好好搞无机，而是一直沉溺于Clayden中无法自拔（刚开始我甚至还在看邢大本，幸好改成Clayden了）。初赛的有机一年难过一年，对于第一年开始的考生，其实最有效的办法是在无机中多抢点分，有机则无需多看，懂点基本的就行。这时我也明确感觉到了我课内成绩的下降，在市统测时尤为明显。我心中有好几个原因，竞赛也是其中一个，不知道哪个才是主要的。

暑假的高强度训练容易让人失去对化学的兴趣。我必须得坦白，人是会累的。虽然我知道二中这种做六休一的模式相比学军等学校已经好太多了，但是或许我不是学化学的料吧，在最后几天我已经很难继续吸收更多知识了，上课也开始迷离。至于暑假的培训，二中总是不听从学生的意见，一味往偏难怪的机构里塞人，哪怕是我去的浙大夏令营也必须去精品班，完全忽略了像我们这种人是需要良好的基础这一事实。这里我不得不吐槽一下：海亮的食堂是真的烂，寝室竟然没有阳台。

最后几天，我才较为深刻地、系统地补习了一部分物化、结构和无机，并继续看了一会中有。可惜这句古话不灵验了，亡羊补牢为时已晚啊。开学时的几天还要溜回教室上课，开学前还要补暑假作业。应该说，最后几天我的水平有一定提升，但总体已经定型了。（至于考前的冲刺，我只能说，二中老师一般押不中题的嘿嘿，不要问我是怎么知道的）

再一次走进学紫的大门，我才发现，我还是5个月前的我，一点都没变。最后的分数也是在意料之中吧。

在最后的一些话里，我有几个人想深深地感谢：球哥，谢谢您牺牲自己的时间（和谈恋爱时间^_^）无私地给我们上课、讲卷子，关注我们的学习、生活，总是和我们打成一片，我们的成绩不一定好，但是你教我们时，我们很开心；谢谢二中、二钱竞赛团队的各位老师们；谢谢预赛停课时关心我为我录课的韩老师；谢谢二中高一化竞团队的各位同学们，我们遇到问题时总是互帮互助而不是单打独斗；最后还有我的父母，给我买书，帮我做后勤工作，给我支持和鼓励，谢谢！

就这样，我的化竞“生涯”结束了。接下来，我回到高考中去。

2023年9月16日
